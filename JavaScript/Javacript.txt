<script src=""></script>を読み込む位置は、基本的にはbodyタブの最後に書く
→　プログラムは上から順番に実行されるので、全てのHTMLを読み込んだ後にJavascriptのコード読み込まれるようにしたいから
src属性で読み込みたいファイルパスの指定する。
console.log()はメッセージを出力する命令になる。
    ダブルククォーテーション""の中に出力したい内容を書く。
    数値を表示する時は、console.log(5);のように記述する。
    
    また、「+」を文字に結合に使うこともできる。
    console.log("Hello World!!!");
    console.log("Hello" + " World!!!");
    どちらも「Hello World!!!」になる。


Developer Tools(検証ツール)で出力結果を確認
1. 検証ツールを開く
    ・Mac　command + option + i
    ・Windows F12キー
    ・右クリック → 「検証」
2. 検証ツールのconsoleタブを選択する

コメント
    実行結果に影響を与えないけれども、メモ書きとしてコメントを残しておきたい時は、
    「//」と2つ続けると、そこから行末までがコメントになる。

    また、1行だけでなくて複数行にわたって書きたい場合は、「/*」と「*/」の間にコメントを書く。

JavaScriptの基本文法
    プログラムの基本は、「○○は××しなさい」と命令すること。
    今回使ったconsole.log()やwidow.alert()の場合、
    ・「○○は」がconsole, window                     オブジェクト
    ・「××」がlog(), alert()                        メソッド
    ・()の中のが"Hello World"が××の指示に必要な情報     パラメータ
